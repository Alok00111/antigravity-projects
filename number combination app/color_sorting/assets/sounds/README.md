# Audio Assets Required

Please add the following audio files to this directory (`assets/sounds/`):

1. **select.mp3**: Played when selecting a tube.
   - Recommended: A soft pop, click, or water gloop sound.
   
2. **pour.mp3**: Played whilst liquid is pouring.
   - Recommended: A flowing water sound (about 1-2 seconds).

3. **win.mp3**: Played when a level is completed.
   - Recommended: A cheerful victory chime or success jingle.

4. **error.mp3**: Played when a move is invalid.
   - Recommended: A dull thud, buzzer, or glass clink.

5. **bg_music.mp3**: Played in the loop in the background.
   - Recommended: Calm, relaxing ambient music (lo-fi or acoustic).

Ensure filenames match exactly as listed above.
